# gnatolbot

Telegram MVP-бот для первичного сбора обращений к Ирине по гнатологии.

## Что умеет MVP
- Ведёт короткий диалог: запрос → удобное время → телефон → имя.
- Если спрашивают цену консультации, отвечает, что стоимость уточнит администратор в зависимости от клиники, и продолжает сбор заявки.
- Отправляет карточку лида в чат клиники.
- Сохраняет лиды локально в SQLite.
- Опционально экспортирует лиды в Google Sheets (после настройки сервисного аккаунта).

## Быстрый старт
1. Создать `.env` по примеру `.env.example`.
2. Установить зависимости:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
3. Запустить:
   ```bash
   python -m gnatolbot
   ```

## Важно
- Токен бота хранить только в `.env`.
- Так как токен уже был отправлен в чат, после первого теста лучше перевыпустить его через BotFather.
