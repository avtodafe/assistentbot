# gnatolbot

Telegram MVP-бот для первичного сбора обращений к Ирине по гнатологии.

## Что умеет MVP
- Ведёт короткий детерминированный диалог: запрос → удобное время → телефон → имя.
- Можно начать не только с `/start`: первая обычная текстовая реплика тоже запускает сценарий.
- Есть `/help` и `/cancel`.
- На вопросы о цене отвечает фиксированно: консультация стоит 2000 рублей.
- На вопросы о времени не обещает слот: точные варианты предлагает администратор после связи.
- Не ставит диагнозы и не интерпретирует снимки.
- После передачи лида завершает сценарий; при новом сообщении пользователь может начать новый сценарий заново.
- Отправляет карточку лида в чат клиники.
- Сохраняет лиды локально в SQLite.
- Опционально экспортирует лиды в Google Sheets (после настройки сервисного аккаунта).
- LLM используется только как вспомогательный извлекатель структуры, а не как основной генератор клиентских ответов.

## Быстрый старт
1. Создать `.env` по примеру `.env.example`.
2. Установить зависимости:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
3. Запустить:
   ```bash
   python -m gnatolbot
   ```

## Важно
- Токен бота хранить только в `.env`.
- Для запуска на сервере см. `RUNBOOK.md`.
- Так как токен уже был отправлен в чат, после первого теста лучше перевыпустить его через BotFather.
